# Global-employment-unemployment-rates-1991-2025
### Global employment and unemployment by sex and age (1991–2025)


#### About Dataset
This dataset contains global employment and unemployment rates by sex and age group from 1991 to 2025.

The data is sourced from the International Labour Organization (ILO) via the ILOSTAT database and is based on modelled estimates. These estimates combine national data with statistical methods to ensure consistency and comparability across countries and over time.

The dataset has been extracted via the ILOSTAT API and processed for analysis. Only country-level data has been retained, while regional aggregates (e.g. World, income groups, continents) and economic groupings (e.g. G7, BRICS) have been removed. The data has been cleaned, standardized, and structured to ensure consistency across all observations.

##### The dataset includes:

Over 180 countries and territories worldwide
Age groups (15+, 15–24, 25+)
Sex categories (Male, Female, Total)
Annual employment and unemployment rates from 1991 to 2025
Values represent percentages:

Employment rate: share of employed persons in the working-age population
Unemployment rate: share of unemployed persons in the labour force
⚠️ Important Notes
🔹 ILO Data and Methodology
The data is based on ILO modelled estimates. The International Labour Organization collects labour market data from national statistical offices and harmonizes definitions across countries.

Since data availability and methodologies differ between countries, the ILO applies statistical models to:

ensure comparability across countries
fill missing observations
produce consistent time series
As a result, the dataset provides a standardized global view. However, some values—especially for recent years (e.g. 2024–2025)—may be estimated rather than directly observed.

🔹 Countries and Territories
The dataset includes countries and selected territories based on ILO statistical standards.

Some entities (e.g. Western Sahara, Hong Kong, Macao, Taiwan, Palestine) are included as they appear in the original ILOSTAT database. These may not correspond to universally recognized sovereign states but are retained to preserve consistency with the source data.

🔹 Age Groups
The dataset includes the following age groups:

15+ (total working-age population)
15–24 (youth)
25+ (adults)
Note:

The 15+ group is an aggregate and includes both 15–24 and 25+ groups
These categories are not independent and should not be treated as mutually exclusive
🔹 Data Processing
The dataset has been extracted via the ILOSTAT API and processed as follows:

only country-level data has been retained
regional aggregates (e.g. World, income groups, continents) have been removed
economic groupings (e.g. G7, BRICS) have been excluded
duplicates have been removed
data has been standardized and cleaned for analysis
🔹 Data Usage Considerations
values are expressed as percentages (%)
employment and unemployment indicators are comparable across countries
recent years may include modelled values
a small number of country-year combinations may be missing due to data availability
This dataset is suitable for:

global comparisons
time series analysis
economic research
machine learning applications
